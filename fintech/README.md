В проекте используется Java 19 и Gradle 7.6.2  
Внешняя зависимость - Lombok  
# Сборка и запуск  
Для сборки проекта:  
1. Открыть в консоли корень проекта  
2. Выполнить команду  
``gradlew build``  
  
Для запуска проекта:
1. Выполнить команду  
``java -jar ./build/libs/fintech-1.0-SNAPSHOT.jar``
